The Ionic Component Documetation showcases a number of useful components that are included out of the box with Ionic.

Whats is Ionic?

Ionic Framework is an open source SDK that enables developers to build high quality mobile apps with web technologies like HTML, CSS and Javascript.

Whats is Ionic Cloud?

The Ionic Cloud is a platform for managing and scaling Ionic Apps with integrated services like push notifications, native builds, user auth, and live updating.
